# Iron Glass Agenda v1.2

## Novedades de esta versión
- Administrador crea y edita usuarios
- Comisión editable por vendedor
- Vendedor solo ve y edita sus ventas
- Operario y perfiles sin permiso financiero no ven precios ni comisiones
- Agenda mensual visual con detalle por día

## Ejecutar local
```bash
npm install
npm start
```

Luego abre:

```bash
http://localhost:3000
```
